# Renee Rose Tuason - Portfolio

A modern, responsive portfolio website showcasing projects, skills, education, and contact information with light/dark mode support.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Dark Mode** - Toggle between light and dark themes
- **Project Showcase** - Display projects with images, descriptions, and tech stacks
- **Image Gallery** - View project images in a modal gallery
- **Skills Section** - Highlight technical skills with icons
- **Education Timeline** - Display educational background
- **Contact Section** - Easy access to email, social media, and resume download
- **Smooth Animations** - Engaging hover effects and transitions

## Tech Stack

- **Backend**: Python Flask
- **Frontend**: HTML, CSS, JavaScript
- **Database**: MySQL (for MajayjayScholars project)
- **Styling**: Custom CSS with responsive design

## Installation

1. Clone the repository
2. Install dependencies: `pip install flask`
3. Run the application: `python app.py`
4. Open your browser and navigate to `http://localhost:5000`

## Usage

- Navigate through sections using the top menu
- Toggle dark mode with the button in the bottom-right corner
- Click on project cards to view image galleries
- Download resume from the contact section
- Connect via social media links
